Problem:

Implement int sqrt(int x).

Compute and return the square root of x.

Solution:
//
// https://leetcode.com/problems/sqrtx/
//

/**
 * @param {number} x
 * @return {number}
 */
var mySqrt = function (x) {
  return Math.floor(Math.pow(x, 0.5));
};
